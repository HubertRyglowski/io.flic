# Flic for Homey

## Introduction
This app integrate `Flic` into Homey.

This app uses Athom Webhooks and Flic's config on Phone or FlicHub

## Usage ##
1. Install app
2. Add device(s) to Homey.
3. Copy HTTP routes from Device Setting's to Flic's App
